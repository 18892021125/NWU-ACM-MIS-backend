# nwu-acm-mis-backend


## Django install
```
pipenv install django==3.2
```

## Datebase migrate
```
pipenv run python manage.py makemigrations
pipenv run python manage.py migrate
```

## Run 
```
pipenv run python manage.py runserver
```

## Crate superuser
```
pipenv run python manage.py createsuperuser
```
